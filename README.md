## Overview

纸牌游戏，100%为黑白墨水屏设计

-为黑白墨水屏设计的界面

-完整的纸牌游戏

-发一张牌/发三张牌

-计时器/记分板

-撤销功能

-横屏/竖屏切换

适配Android7+，在掌阅Ocean5Pro和Smart X5 Ultra上测试通过
